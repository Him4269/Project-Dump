# Lean Menu

You know lean menu
To use menu *click* the **Left thumbstick down**

## Features

- No Grav (Left Grip)
- Ice Control
- Leave Lobby
- No Clip (Left Top Button)
- ESP
- Lucio (Right Bottom Button)

**New**

- Boosts (Right top and Right Bottom buttons)
- Shut up Gun (Right Grip/Trigger)
- Drift (Left Trigger)
